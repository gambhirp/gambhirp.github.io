Predicting ESG Scores from Financial Data

This project explores how financial metrics and industry context can be used to predict ESG scores of S&P 500 companies. Using Yahoo Finance data and machine learning models like Random Forest and SVM, it identifies sector-specific ESG patterns. The goal is to help investors assess sustainability performance even when ESG disclosures are incomplete.

Bank Churn Prediction & Retention Strategy

A machine learning solution to identify high-risk banking customers using historical demographic, financial, and behavioral data. Random Forest, Logistic Regression, and KNN models were evaluated to uncover key churn drivers and customer segments. The project enables proactive retention strategies by integrating predictive insights into CRM workflows for real-world impact.

Hotel Review Analysis & Sentiment Benchmarking

This project analyzes over 515,000 hotel reviews from Booking.com to extract insights using NLP, sentiment analysis, and topic modeling. It classifies guest sentiment, identifies common praise and complaints, and benchmarks hotel performance using visualizations and GPT-based summaries enabling hotel providers to improve service and guest satisfaction.
